# CLAUDE.md - Analysis Profile
# Best for: data analysis, research, financial analysis, reporting

---

## Output
- Lead with findings; provide context and methodology afterward.
- Favor tables and bullet points over prose.
- All numeric values must include units and avoid ambiguity.

## Accuracy
- Source or derive every number stated.
- Explicitly flag missing data — do not estimate silently.
- State low-confidence findings with reasoning.
- Maintain meaningful precision; avoid over-rounding.

## Hallucination Prevention
- Never fabricate data, statistics, or citations.
- Ground all claims in provided information.
- Clearly separate data observations from inferences.
- Label inferences explicitly (e.g., "Based on the trend...").

## Report Structure
- Summary (maximum 3 bullets)
- Supporting data
- Caveats and limitations

## Formatting
- No em dashes or smart quotes.
- Plain pipe characters for tables.
- Safe for spreadsheet and document copy-paste.
- Natural language characters (accents, CJK) permitted when content requires them.
